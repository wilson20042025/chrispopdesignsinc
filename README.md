# Chris Pop Design Inc.

A precision-focused architectural studio website built with Next.js, Tailwind CSS 4, and TypeScript.

## Project Structure

- `app/` - Next.js App Router directory
  - `components/` - Shared UI components
  - `projects/` - Archive of selected architectural works
  - `philosophy/` - The studio's ethos and principles
  - `contact/` - Global inquiry system
  - `globals.css` - Custom Tailwind 4 theme configuration
  - `layout.tsx` - Root layout with Inter font and Material Symbols

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Type Safety**: [TypeScript](https://www.typescriptlang.org/)
- **Fonts**: Inter & Material Symbols Outlined (Google Fonts)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) to view the project.
